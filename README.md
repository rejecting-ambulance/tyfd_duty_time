# tyfd_duty_time
Automatic uplode the duty time of a signed month.


# How to use

>使用請參閱打基準表vvcd.xlsm中的黃框說明。  
>本文件說明 Excel 的3個分頁：**日曆_輸入區**、**假別_設定區**、**開發歷程**  

## 日曆_輸入區
1. 輸入基本資料：帳號、密碼、姓名、要登打的年月。
2. 類機關組別：安檢1~4、外勤組別：外勤A、B班，選擇右方會更新，選好按過去複製到左方班別。
3. 模式：登打、清起來，後者用來刪除已登打的資料。
4. 最後再補上彈性休息時間(時段A起、時段A迄)兩欄。

## 假別_設定區
1. 初始假別已有設定，有需求可新增或修改原設定。
2. 登打名稱為各區塊左上儲存格，右方四欄分別為**假別單位**、**項目**、**時段起**、**時段迄**。
3. 時段起訖以數字設定為固定時間，輸入A則會讀取**日曆_輸入區**起訖時間。
4. 可以登打的Lable會出現在**日曆_輸入區**中間的彩色區塊(J15:K21)。


# Additional problem
### Excel 巨集無法使用
1. 複製O欄班別，以值貼上到C欄。
2. 關閉 Excel 後，執行打基準表程式。
